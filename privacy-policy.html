Privacy Policy for TraceFence

TraceFence 隐私政策

TraceFence is designed to keep core product data on the user's device whenever possible.

TraceFence 的设计目标是尽量把核心产品数据保留在用户设备上。

  • local monitoring and summary data stays on device by default本地监控与摘要数据默认保留在本机
  • optional services must be opt-in可选服务必须由用户主动开启
  • license metadata may be stored locally if licensing is enabled如果启用授权，授权元数据可能会本地保存
  • the latest version of this policy lives in this repository本政策的最新版本会保存在这个仓库中

For the full legal page, this repository also exposes the HTML version at the root.

完整法律页面也会以根目录 HTML 的形式提供，方便官网和 GitHub Pages 直接引用。
