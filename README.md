# TraceFence

TraceFence is a local-first command center for understanding and controlling AI coding agents on your computer.

This repository is the public home for TraceFence documentation, legal pages, and downloadable installers. The website can safely link here as the canonical download and support hub.

## Downloads

Official macOS and Windows installers are published through this repository’s [Releases](https://github.com/AI-Scarlett/TraceFence/releases) page.

- Windows: `TraceFenceSetup-<version>.exe`
- macOS: `.dmg` or `.zip`

## What TraceFence provides

- AI agent activity overview
- token and context analytics
- command confirmation and approval workflows
- process and deep monitoring
- local audit history and report export
- multiple appearance and color themes
