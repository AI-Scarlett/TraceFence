●GitHub ReleasesGitHub 发布页

Publish macOS and Windows installers here.

macOS 和 Windows 安装包以后都从这里发。

Use GitHub Releases for public installers, checksums, and release notes. The website can point users here for the latest download.

用 GitHub Releases 发布安装包、校验值和更新说明。官网可以直接把用户导到这里下载最新版本。

Suggested assets建议的发布内容

  • macOS `.dmg` or `.zip`macOS `.dmg` 或 `.zip`
  • Windows `.exe` or `.msi`Windows `.exe` 或 `.msi`
  • release notes更新说明
  • checksum file if available如有则附校验文件

Distribution rules分发规则

  • Keep versioning visible in release names.版本号要在发布名里一眼能看见。
  • Keep direct-download builds separate from App Store builds.直下载版本和上架版本分开管理。
  • Link the website to this repository and its releases.官网链接到这个仓库和 Releases。
