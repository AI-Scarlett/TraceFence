Install TraceFence

安装 TraceFence

  1. Open the latest GitHub Release.打开最新的 GitHub Release。
  2. Download the macOS or Windows installer.下载 macOS 或 Windows 安装包。
  3. Open the package and complete the OS prompts.打开安装包并完成系统提示。
  4. Launch TraceFence and allow any first-run permissions.启动 TraceFence，并允许首次运行需要的权限。
