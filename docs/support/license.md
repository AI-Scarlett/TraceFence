# License Guide

TraceFence direct-download builds may use license keys or another purchase verification method.

## What users should expect

- activation happens locally in the app
- the app should tell users when a key is valid or expired
- license checks should not break basic onboarding or leave the UI blank
- trial or grace periods must be clearly explained

## What this repo should document

When licensing is enabled, the public repo should explain:

- where to buy
- how to activate
- how many devices are included
- what happens when a license expires
- how to contact support if activation fails
